About

What is it?

EZRP is an online based Enterprise Resource Planning application which was designed in an open source platform.
The open source platform for this project was inline with company requirements for heavy customisation and so the base
version of this application is designed to be relatively generic and easy to use.

The base version focuses on five main modules:
- Human Resources
- Parties (Customers & Vendors)
- Orders (Sales & Purchase)
- Products
- Stock
